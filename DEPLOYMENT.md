# FractionDev Pro - Deployment Guide

## Overview
This is a high-end, static web application designed for the educational mathematics niche. It features a reactive calculation engine, interactive quiz modules, visual fraction bars, and multi-language support (English/Japanese).

## Key Features
- **Reciprocal Trace**: Shows the "Flip and Multiply" conversion steps.
- **Visual Lab**: Bar graphs to represent fraction proportions.
- **Interactive Quiz**: Random problem generator with automatic verification.
- **Math History**: Remembers the last 10 calculations using 100% client-side `localStorage`.
- **Dark Mode & Print**: Pro tools for students and teachers.

## Technical Architecture
- **No Backend Required**: 100% Static HTML/CSS/JS.
- **Responsive**: Mobile-first design using Flexbox and Grid.
- **Privacy First**: No user data is sent to any server.

## Deployment Steps
1. Push to GitHub.
2. Connect to Vercel/Netlify.
3. Done.
